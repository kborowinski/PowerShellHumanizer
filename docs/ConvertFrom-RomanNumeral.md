# ConvertFrom-RomanNumeral

Convert a Roman Numeral String to an Int.

## Parameters

### Parameter Set 1

- `[String[]]` **RomanNumeral** _One or more Roman Numeral Strings to convert._ Mandatory, ValueFromPipeline

## Examples

### Example 1



```powershell
ConvertFrom-RomanNumeral IV
4
```

## Outputs

- `int`
