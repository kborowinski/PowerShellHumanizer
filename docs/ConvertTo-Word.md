# ConvertTo-Word

Convert an Int to a String of one or more words.

## Parameters

### Parameter Set 1

- `[Int32[]]` **Number** _One or more Ints to convert._ Mandatory, ValueFromPipeline

## Examples

### Example 1



```powershell
1337 | ConvertTo-Word
one thousand three hundred and thirty-seven
```

## Outputs

- `string`
